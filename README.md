## 技术栈
### 核心技术
- **Vue 2.x/3.x**：组件化开发与数据驱动
- **Vue Router**：单页应用路由管理
- **JavaScript (ES6+)**：交互逻辑与数据处理
- **CSS3**：响应式布局与过渡动画
- **本地静态数据**：src/data 目录提供模拟数据

### 开发工具
- **Node.js + npm**：依赖管理与项目构建
- **VS Code**：代码编辑器（推荐插件：Vetur、Prettier、ESLint）
- **Chrome DevTools**：调试与性能优化

## 环境配置
### 开发环境要求
- 操作系统：Windows 10+/macOS 12+/Linux
- Node.js：v14.0.0 或更高版本
- 浏览器：Chrome 90+、Edge 90+

### 环境搭建步骤
```bash
# 1. 克隆项目
git clone [https://github.com/biqibaomiaomiaowu/Honor-of-Kings-website.git]
cd honor-of-kings-vue

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run serve

# 4. 访问项目
# 在浏览器打开 http://localhost:8080（端口以终端输出为准）
```

## 功能模块说明
### 核心功能
1. **首页展示**：轮播图、热门英雄/装备推荐、快速导航入口
2. **英雄模块**：英雄列表展示、英雄详情页（技能、属性）
3. **装备模块**：装备分类展示、装备详情页（属性、合成路径）
4. **地图模块**：地图介绍、野怪分布、资源点说明
5. **世界观模块**：王者大陆背景故事、阵营介绍
6. **公共交互**：导航栏路由跳转、页脚信息展示、回到顶部按钮

---

## 📖 使用说明
1. 启动项目后，通过顶部导航栏可进入「英雄」「装备」「地图」「世界观」等页面。
2. 英雄列表页点击英雄卡片，可进入详情页查看技能、属性等信息。
3. 装备页支持按「攻击」「防御」「法术」等分类筛选装备。
4. 地图页可以通过点击热点了解野怪属性及游戏机制
5. 世界观介绍了王者世界的背景故事
6. 页面滚动到底部时，点击右下角「回到顶部」按钮可快速返回顶部。

## 注意事项
1. **代码规范**：遵循 Vue 官方风格指南，变量使用小驼峰命名，组件名使用大驼峰命名。
2. **数据维护**：新增/修改数据时，需同步更新 src/data 下对应模块的 JS 文件。
3. **兼容性**：确保页面在 Chrome、Edge 浏览器中正常显示，适配 1280px 以上屏幕宽度。
4. **团队协作**：每日提交代码至个人分支，遇到冲突及时沟通解决。
