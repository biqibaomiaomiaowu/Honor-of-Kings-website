####  迁移开发指南
*   **语法标准**：统一使用 Vue 3 `<script setup>` 语法。
*   **样式迁移**：原 HTML 中的 CSS 放入组件的 `<style scoped>` 中。
*   **资源引用**：图片路径从 `./imgs/xxx` 修改为 `/imgs/xxx`。
*   **根元素规则**：虽然 Vue 3 支持多根节点，但为了兼容性和避免 ESLint 报错（如 `[vue/no-multiple-template-root]`），建议每个组件的 `<template>` 最外层仍保留一个 `<div>` 包裹。
